## Drawing Board
It is the WebGL app.

To start:
- npm install
- npm start
- localhost:8080

The app includes 2 tabs:
  * Drawing board - WebGL 2d with a few filters
  * Planet - WebGL 3d. You can:
    - rotate the planet
    - change the light position
    - change water covreage
    - change mountains height
    - change the planet's colors
    - implement fxaa
